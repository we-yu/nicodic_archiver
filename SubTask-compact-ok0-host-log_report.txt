OK0 compression rule
- One-line [STEP OK0 🟢] is emitted only when all of these are true:
  status=success, stored_new=0, reason resolves to already_up_to_date,
  no WARN/FAIL detail was emitted, exactly one successful page was observed,
  saved_after is known, observed_after is known, and no status text is
  present.
- The compact OK0 line includes step, article_id, title, saved, observed,
  page, elapsed, and reason.
- URL is omitted from the compact OK0 line.
- Non-qualifying targets fall back to the existing detailed compact format.
